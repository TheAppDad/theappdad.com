# Chosen Legal Pages – Deploy to GitHub

These files match the design of **theappdad.com** so the Privacy, Support, and Terms pages look like part of your main site when linked from there.

## Deploy to your Chosen-Right-App repo

1. Copy these files into the **root** of your [Chosen-Right-App](https://github.com/TheAppDad/Chosen-Right-App) repo:
   - `index.html` (overwrites existing – this is your Privacy Policy)
   - `support.html` (overwrites existing)
   - `terms.html` (overwrites existing)
   - `styles.css` (add this file)

2. Commit and push to GitHub.

3. GitHub Pages will serve them at:
   - https://theappdad.github.io/Chosen-Right-App/ (Privacy)
   - https://theappdad.github.io/Chosen-Right-App/support.html
   - https://theappdad.github.io/Chosen-Right-App/terms.html

## Notes

- The header links back to **theappdad.com** so users can return to your main site.
